UE5 Material — UI 모션

Interaction Wave

UE5 Material을 사용하여 텍스처 모양에 따라 인터렉션되는 파장을 생성하는 노드구현

노드 구현 포인트

텍스처의 모양이 특징이 되어 게임의 아이덴티티가 녹아들수 있게 만든다.

반복되며 속도 조절이 가능해야 한다.

노드 풀이

STEP 1

VetorToRadialValue 방사형으로 퍼지는 패턴을 만들기 위한 기본 값 제공

STEP2

Power 노드를 사용하여 중심 근처 값은 0에 가까워지고

외곽 값은 더 빠르게 증가중심을 더욱 선명하게 보이게 하며

마스크 효과를 강화

STEP3

Subtract 노드를 사용하여 웨이브가 나타나는 지점을 조정하는 역할

결과적으로 원형 마스크를 특정 위치에서 보이게 만들어줍니다.

STEP4

1-x를 통해 웨이브를 반전 시켜줍니다.

STEP5

Sine 노드를 일정한 주기로 반복되는 원형 패턴을 만들어 줍니다.

자주 사용하게 되는 노드 설명

VectorToRadialValue : 중심에서의 거리 값을 계산하는 노드로, 이를 이용해 방사형 UI 효과, 마스크, 애니메이션을 만들 때 사용

Mask : 특정 색상 채널(R, G, B, A)만 선택할 때 사용

Power : 0~1의 강도를 조절하여 이미지의 여백을 조절해줄수 있습니다.

텍스처 타일링

Append : 입력된 개별 스칼라 또는 벡터 값을 하나의 벡터로 결합
