UE5 Material — 무기 축복 연출

Shape Wave

UE5 Material을 사용하여 텍스처 모양을 합성하여 다양한 웨이브 효과 노드

노드 구현 포인트

UV 좌표 변형 및 격자 생성

원형 마스크 생성

텍스처와 결합하여 최종 효과 생성

노드 풀이

텍스처 좌표 변환 및 샘플링

TexCoord → Multiply → Floor → Divide → Subtract(0.5)

TexCoord (Texture Coordinate): 기본적인 UV 좌표를 제공.

Multiply: UV 좌표를 특정 값(PixelNum)과 곱해 샘플링 해상도를 조정.

Floor: 좌표를 정수 단위로 변환해, 격자 기반 효과를 만들기 위한 사전 처리.

Texture Sample: 텍스처를 적용하고, 특정 효과에 사용할 수 있도록 값을 가져옴.

Power(X, 1): 텍스처 값을 조정해 대비를 조절.

원형 마스크 효과 생성

Length → Multiply(WavePower) → Sine → Multiply(0.5)

Subtract(0.5): UV 좌표를 중심(0,0) 기준으로 변환.

Length: 각 픽셀의 거리 값을 계산하여 방사형(원형) 그라데이션을 생성.

Multiply (WavePower): 거리 값을 조정하여 원형의 크기를 조절.

Sine: 거리 값을 활용하여 웨이브(파형) 효과를 생성.

SmoothStep: 특정 범위의 값을 0~1 사이로 부드럽게 변환하여, 마스크 효과를 부드럽게 만듦.

애니메이션을 위한 시간 기반 변형

Time → Multiply(Speed) → Frac

Time: 시간 값을 입력받아 애니메이션 효과를 주기 위한 입력 데이터.

Multiply (Speed): 시간 값과 속도 값을 곱해 파형 애니메이션의 속도를 조절.

Frac: 시간 값의 소수 부분만 사용하여 반복적인 패턴을 생성.

최종 효과 결합 및 조정

Texture Sample → Multiply → Lerp

Add: 여러 개의 효과를 합쳐서 하나의 출력값을 만듦.

Lerp: 두 개의 값을 블렌딩하여 부드러운 전환 효과를 적용.

Multiply: 마스크를 적용하여 특정 영역만 표시.
